# 文章发布系统

## 项目简介
文章发布系统是基于Java语言、SSM框架、JSP页面技术、Mysql数据库的发布系统。分为前台系统和后台系统。前台系统负责展示阅读文章，后台系统负责发布管理文章、用户、通告等。模块包含：1.用户模块 2.文章模块 3.公告管理 4.评论管理 5、留言管理 适合作为毕设设计、课程设计。

## 技术介绍
Java Web、JSP页面、SSM框架、Mysql数据库

## 开发工具
Eclipse或者MyEclipse或者IDEA、Navicat

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
